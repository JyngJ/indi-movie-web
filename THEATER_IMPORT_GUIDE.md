## 🛠️ 추가 방법

### 방법 1: 어드민 콘솔에서 수동 추가 (권장 - 즉시 반영)

1. `/admin` 페이지 접속 → **관리 탭** 클릭
2. **새 극장** 버튼 클릭
3. 다음 정보 입력:
   - 극장명: e.g., "강화작은영화관"
   - 도시: "인천" / "경기" / "강원"
   - 주소: 전체 주소 입력
   - 위도/경도: 구글맵에서 복사
   - 전화/웹사이트: 선택사항
4. **극장 생성** 클릭

### 방법 2: API 스크립트로 일괄 추가 (개발자용)

```bash
# 사전 준비: 좌표 정확도 검증 필요
# src/data/theaters-to-add.ts 의 lat/lng 를 구글맵 API로 업데이트

# 또는 다음 명령으로 직접 추가
curl -X POST http://localhost:3000/api/admin/theaters \
  -H "Content-Type: application/json" \
  -H "Cookie: admin_session=YOUR_SESSION" \
  -d '{
    "name": "강화작은영화관",
    "city": "인천",
    "address": "인천광역시 강화군 고비고개로 19번길 12 강화문예회관 2층",
    "lat": 37.7408,
    "lng": 126.5922
  }'
```
